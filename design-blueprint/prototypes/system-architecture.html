Why precompute?

Flood and climate data are heavy rasters. Aggregating them into cells once — not per request — is what keeps the app fast enough to feel calm.

Weights stay personal

Sub-scores are universal and cached; only the weighted aggregation runs per user. The same property scores differently for different people, cheaply.

Confidence travels with data

Every cached value carries its source tier and confidence, so the UI can stay honest about Nigeria's uneven coverage without extra lookups.

Built to scale out

New country = new source adapters + regional thresholds. Layers 3–6 don't change, so global expansion is mostly a Layer 1–2 effort.
